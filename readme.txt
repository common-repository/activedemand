== Description ==

Adds the <a href="https://www2.activedemand.com/s/vaiXT">ActiveDEMAND</a> tracking script to your website. As well this plugin gives you the ability to use shortcodes to embed ActiveDEMAND webforms into your widgets, pages, posts, and sidebars.

== Installation ==

1. Install the plugin from within the Dashboard or upload the directory `activedemand` and all its contents to the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress
